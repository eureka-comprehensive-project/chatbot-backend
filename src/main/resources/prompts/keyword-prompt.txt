지금까지의 대화를 기반으로 키워드를 추출해 주세요.


아래 형식 그대로 String 키워드 단어 하나로만 응답하세요. 설명은 필요 없습니다.

지금까지의 대화에서 언급된 단 하나의 키워드를 응답하세요.

아무 설명 필요 없습니다.

예시 1: 청소년
예시 2: 노인
예시 3: 유아



사용자의 응답(피드백)을 받은 후, 다음 JSON 형식으로 다시 정리해 출력하세요:

절대 준수 사항
- 각각의 코드는 다음과 같아요
    - 0 : 만족하는 경우
    - 1 : 데이터 부족
    - 2 : 데이터 너무 많음
    - 3 : 가격 비쌈
    - 4 : 가격 너무 저렴
    - 5 : 부가혜택 부족
    - 6 : 피드백 종료
- 당신은 위 6가지 경우 중 사용자가 어떤 피드백을 보내는 건지 알아내야해요.
- 어떤 경우인지 판단이 끝났다면, 다음과 같은 형태로 code를 넣어서 json으로 반환하세요 그 이후에는 아무말도 하지 마세요
    예시 1: 사용자가 데이터가 너무 부족하다고 했을 경우
                {
                    "feedbackCode" : 1
                }

    예시 2:  사용자가 가격이 비싸다고 했을 경우
                {
                    "feedbackCode" : 3
                }
    예시 3:  사용자가 "끝" 이라고 했을 경우
                    {
                        "feedbackCode" : 6
                    }
